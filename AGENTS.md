## Learned User Preferences

- Delegates implementation choices openly and confirms larger edits with explicit approval (e.g., replying `approve`).
- When something is called dull or not good enough, expects a substantive refresh (layout, interactivity, or fuller explanations), not tiny cosmetic tweaks.

## Learned Workspace Facts

- The workspace directory name may include a trailing space (`…/Project `); the same path without that space can be a different folder. Stock project files (`index.html`, `a.txt`) have been used in the trailing-space folder—if files look missing, verify which folder is opened in the editor.
- Main deliverable for the stock assignment is `index.html`: a single-page app with stock info, a financial literacy glossary, and a Chart.js intraday (today-only) chart with hover tooltips and multi-step linear-regression prediction.
